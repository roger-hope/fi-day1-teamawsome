Was ist eine Class in Java?

In Java haben Klassen drei Hauptaufgaben:
1. Klassen legen die Struktur von Objekten fest.
2. Klassen bestimmen das Verhalten von Objekten.
3. Klassen werden zum Erzeugen von Objekten verwendet.
Ein Objekt, das von einer Klasse erzeugt wurde, nennt man eine Instanz. (Quelle: A. Hölzl, Matthias, Allaithy Raed, and Martin Wirsing. Java kompakt: Eine Einführung in die Software-Entwicklung mit Java. Springer-Verlag Berlin Heidelberg, 2013.)

public class Person {
 // Attribute
 String name;
 int alter;

 // Konstruktor
 public Person(String name, int alter) {
  this.name = name;
  this.alter = alter;
 }

 // Methoden
 public void printInfo() {
  System.out.println("Name: " + name);
  System.out.println("Alter: " + alter);
  }
 }

Was ist ein Constructor in Java?

Konstruktoren dienen zum Initialisieren von neu erzeugten Objekten. Genau wie Methoden konnen sie Parameter haben; einen Konstruktor ohne Parameter nennt ¨ man auch Standardkonstruktor. Wenn kein Konstruktor definiert wird, erzeugt Java automatisch einen Standardkonstruktor. (Quelle: A. Hölzl, Matthias, Allaithy Raed, and Martin Wirsing. Java kompakt: Eine Einführung in die Software-Entwicklung mit Java, S. 80, Springer-Verlag Berlin Heidelberg, 2013.)

// Konstruktor aufrufen
Person person1 = new Person("Thomas", 30);

Was ist eine Instanz in Java?

Ein Objekt, das die von einer Klasse A vorgegebene "Form" hat, nennt man Instanz der Klasse A oder Objekt vom Typ A. Zum Erzeugen von Instanzen wird ein Konstruktor der entsprechenden Klasse verwendet. Jede Instanz verfügt über Speicherplatz für alle Instanzvariablen ihrer Klasse und kann alle Methoden dieser Klasse zum Manipulieren ihrer Daten ausführen. Quelle: A. Hölzl, Matthias, Allaithy Raed, and Martin Wirsing. Java kompakt: Eine Einführung in die Software-Entwicklung mit Java, S. 9, Springer-Verlag Berlin Heidelberg, 2013.)

// Neue Instanz der Klasse Person erstellen
Person person1 = new Person("Thomas", 30);

Was ist eine Methode in Java?

Eine Methode ist ein Codeblock, der nur ausgeführt wird, wenn er aufgerufen wird. Sie können Daten, sogenannte Parameter, an eine Methode übergeben. Methoden werden verwendet, um bestimmte Aktionen auszuführen, und sie werden auch als Funktionen bezeichnet. Warum Methoden verwenden? Um Code wiederzuverwenden: Sie definieren den Code einmal und können ihn dann mehrfach verwenden.

// Methode aufrufen
person1.printInfo();
